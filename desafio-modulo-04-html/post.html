Aprendendo a construir o esqueleto de um site

Importancia do HTML

O HTML é uma linguagem de marcação que permite estruturar todo o site ou pagina da web, é por meio dele que você adiciona a maioria dos elementos que vão estar presentes no site como, titulos, subtitulos, paragrafos, listas, imagens, links e muitos outros elementos.

Toda essa estruturação é a base para os sites e onde voce criará um esqueleto para o site para posteriormente adcionar estilos com o css e interação com o Java Script, compondo assim um website.
